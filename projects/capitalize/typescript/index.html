Capitalize in Typescript

Published on 03 October 2019 (Updated: 03 October 2019)

Welcome to the Capitalize in Typescript page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

if (process.argv.length < 3 || process.argv[2].length < 1) {
    console.log("Usage: please provide a string")
    process.exit(0)
}

let myStr: string = process.argv[2];

const capitalize = (str: string) => str[0].toUpperCase() + str.slice(1);

console.log(capitalize(myStr));
