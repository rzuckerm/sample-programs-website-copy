Even Odd in Julia

Published on 01 November 2021 (Updated: 01 November 2021)

Welcome to the Even Odd in Julia page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution


#!/usr/bin/julia

function err() 
   println("Usage: please input a number")
end

function even_odd(n)

    if (n % 2 == 0 )
      return "Even"
    else
      return "Odd"
    end  
end

try
    println(even_odd(parse(Int, ARGS[1])))
catch e
    err()
end
