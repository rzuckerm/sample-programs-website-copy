Factorial in Bash

Published on 09 October 2019 (Updated: 09 October 2019)

Welcome to the Factorial in Bash page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

#!/bin/bash

number=$1

if [[ -z "$number" ]]
then
    echo "Usage: please input a non-negative integer"
    exit 1
fi

if ! [[ "$number" =~ ^[0-9]+$ ]]  
then 
        echo "Usage: please input a non-negative integer"
        exit 1
fi

value=1

while [ $number -gt 1 ]
do

  value=$((value * number))
 
  number=$((number - 1))

done

echo $value
