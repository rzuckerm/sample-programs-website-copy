Fizz Buzz in Haskell

Published on 13 August 2018 (Updated: 13 August 2018)

Welcome to the Fizz Buzz in Haskell page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

module Main where

fizzbuzz :: Int -> String
fizzbuzz x
    | x `mod` 15 == 0 = "FizzBuzz"
    | x `mod` 3  == 0 = "Fizz"
    | x `mod` 5  == 0 = "Buzz"
    | otherwise       = show x

main = mapM (putStrLn . fizzbuzz) [1..100]
