Reverse String in Euphoria

Published on 19 February 2023 (Updated: 19 February 2023)

Welcome to the Reverse String in Euphoria page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

include std/io.e
include std/sequence.e

-- Parse 1st command-line argument
sequence argv = command_line()
sequence s = ""
if length(argv) >= 4
then
  s = argv[4]
end if

-- Show reversed string
sequence t = reverse(s)
printf(STDOUT, "%s\n", {t})
