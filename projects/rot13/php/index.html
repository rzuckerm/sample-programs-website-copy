Rot13 in Php

Published on 13 October 2019 (Updated: 13 October 2019)

Welcome to the Rot13 in Php page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

<?php

if (empty($argv[1]) || $argv[1] == "" || count($argv) == 0) {
    die("Usage: please provide a string to encrypt\n");
}

function rot13(string $string)
{
    return strtr($string,
        'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz',
        'NOPQRSTUVWXYZABCDEFGHIJKLMnopqrstuvwxyzabcdefghijklm');
}

echo rot13($argv[1]) . "\n";
