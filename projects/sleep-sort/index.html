Sleep Sort in Every Language

Published on 08 October 2019 (Updated: 02 May 2020)

Welcome to the Sleep Sort page! Here, you'll find a description of the project as well as a list of sample programs written in various languages.

Description

Sleep sort is a sorting algorithm that for each input numeric variable it starts a thread (or thread like process) that automatically goes to sleep for given number of time units, eg. seconds. When a thread complets, it returns the number. When all threads complete, a sorted array of numbers is returned.

Performance

Time complexity of this algorithm strictly depends on value of the highest number, therefore it's always O(n), space complexity is dependent on numbers of threads started - O(n) where n is number of inputs.

Requirements

Write a sample program that takes a list of numbers in the format "4, 5, 3, 1, 2". It should then sort the numbers and output them:

$ ./sleep-sort.lang "4, 5, 3, 1, 2"
1, 2, 3, 4, 5

Testing

Every project in the Sample Programs repo should be tested. In this section, we specify the set of tests specific to Sleep Sort. In order to keep things simple, we split up the testing as follows:

  • Sleep Sort Valid Tests
  • Sleep Sort Invalid Tests
